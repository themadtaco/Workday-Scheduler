# Workday-Scheduler

## Purpose
Develop website so that users can add and save timeblocks on a dynamic scheduler. 
Timeblocks change color based on whether they are in the past, present, or future.

## Built With:
* HTML
* CSS
* Javascript & JQuery
* Luxon API
